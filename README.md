# Atlas_Compiler

## Description
The Atlas Compiler is an independent project, and is aimed primarily at cultivating a deeper understanding of how great languages like C, C++, Java, etc are used to run technology in the modern age    

## Structure 
#### Consists of:
- Lexer (Tokenizer)
- Parser
- Code Emitter
- "Helper" classes
